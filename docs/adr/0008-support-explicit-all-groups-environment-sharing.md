# Support explicit all-groups environment sharing

> **Status: Superseded.** Environments are now tenant-wide and have no group-sharing modes. See [Tenant Access](../tenant-access.md).

Environments may be private to their owner group, shared with selected groups, or explicitly shared with all current and future non-archived access groups. All-groups sharing is not the default and must be presented in the UI with a warning that it grants every group permission to run agents against the environment's codebase without granting environment management authority.
